Vue is a progressive framework for building user interfaces.
Vue.js is a JavaScript front-end framework that was built to organize and simplify web development
It is designed from the ground up to be incrementally adoptable. 
It is easy to pick up and integrate with other libraries or existing projects. 
Vue uses an HTML-based template
Vue templates are valid HTML that can be parsed by spec-compliant browsers and HTML parsers
Components are one of the most powerful features of Vue
 In components it is necessary to divide the whole app into small, self-contained, and often reusable components to make development manageable
Components extend basic HTML elements to encapsulate reusable code
